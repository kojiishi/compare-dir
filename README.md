# compare-dir

Compare two directories and show the differences.

## Installation

```bash
cargo install compare-dir
```

If you want to install from the local source code:
```bash
cargo install --path .
```

## Usage

```bash
compare-dir <dir1> <dir2>
```

## Examples

```bash
compare-dir dir1 dir2
```
